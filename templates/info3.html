• Al iniciar, se abrirá una sesión de Whatsap web en una nueva ventana de Chrome.


  • El tiempo de espera para escanear el código QR es de 20 segundos.


  • Una vez activado el QR, comienza el evío de mensajes.


  • Al terminar, se cerrará la sesión de Whatsapp Web y se habilitará la opción para eliminar los contactos agendados.
